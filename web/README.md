# ShadowTest Frontend 🎨

The high-performance monitoring dashboard for ShadowTest simulations. Built with a "Midnight Gold" aesthetic to provide a premium, enterprise-grade experience.

## ✨ Features

- **Live Trace Visualization**: Watch AI personas reason and move through your product in real-time.
- **Persona Management**: Configure behavioral traits and technical levels for synthetic users.
- **Simulation Control**: Define complex tasks and launch autonomous testing agents.
- **Seamless Auth**: Integrated with **Clerk** for secure, user-isolated sessions.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4 (Custom Midnight Theme)
- **UI Components**: Shadcn/UI & Radix UI
- **Authentication**: Clerk
- **Icons**: Lucide React
- **Client**: Prisma Client (via API)

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Environment Variables**:
    Copy `.env.local.example` to `.env.local` and add your Clerk keys.
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
